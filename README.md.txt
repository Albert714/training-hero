# Тренировка героя

Небольшой консольный проект на C#, демонстрирующий работу с основами языка: переменные, условия, циклы (`for`, `while`, `do while`), а также базовую структуру программы.

## Описание

Программа имитирует "зал прокачки", куда пользователь может попасть после ввода пароля. После успешной авторизации предоставляется выбор из трёх видов "тренировок":

- **Выносливость** — серия приседаний с циклом `for`
- **Сила** — отжимания с использованием `while`
- **Интеллект** — таблица умножения на 2 и 3 с использованием `do while`

Проект реализован в рамках обучения C# как часть закрепления базовых конструкций языка.

## Используемые конструкции

- `if / else if / else`
- `switch`
- `for`, `while`, `do while`
- `bool`, `int`, `string`
- Преобразование строк в числа (`Convert.ToInt32`)
- Вложенные циклы

## Структура

Код разделён на блоки:
1. Проверка пароля
2. Основное меню
3. Обработка выбора пользователя
4. Завершение работы

## Цель проекта

Укрепить навыки работы с базовой логикой на C# и подготовиться к изучению следующих тем: массивы, методы и основы ООП.

## Как запустить

Проект написан на C# и может быть запущен в любой среде, поддерживающей .NET:

- Visual Studio
- Rider
- VS Code (с плагином C#)
- CLI: `dotnet run`
